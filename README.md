## 平台支持

|    OS     |    GRAPHICS API    |
| :-------: | :----------------: |
| `Windows` | `OpenGL`& `Vulkan` |
|  `MacOS`  | `OpenGL`& `Vulkan` |
|   `Ios`   | `OpenGL`& `Vulkan` |
| `Android` | `OpenGL`& `Vulkan` |
|   `Web`   |      `OpenGL`      |

注：安卓支持`SurfaceView`、`AHarwareBuffer`两种上屏方式
